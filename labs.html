Keenul's Lab Assignments

Keenul has worked on a handful of labs throughout the course of the semester. Check out screenshots of the code and final products below!

Lab 1 Assignment

In the first lab of the class, the goal was to learn the basics of HTML by playing around with text and image insertion.

Lab 2 Assignment

In this lab, Keenul learned how to create multiple pages and use CSS stylesheets to dress up the dog adoption website she created. She also learned how to embed links onto the web page.

Lab 4 Assignment

In lab 4, Keenul continues to apply basic HTML and CSS style sheets. For this website, she learned how to use columns to organize information. She also used media queries to make the website compatible across all devices. Styling print sheets was also applied to this lab.

Lab 6 Assignment

In this lab, Keenul learned how to create an HTML form while using a CSS stylesheet to make it look more appealing.
